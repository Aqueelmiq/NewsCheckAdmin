Instructions

Login To Your Account

Installation:

  • Download Chrome Extension
  • Go To Chrome -> Settings -> Extensions
  • Click Load Unpacked Extension
  • Navigate to Root Folder of The chrome extension

    • Usage:

  • Go To facebook.com and login to your account
  • You should notice a colored progress bar.
  • Red Means News is likely fake or very biased.
  • Orange Means News is unlikely to be fake, but use caution since source is not reputed.
  • Green Means News is not fake and is from a reputed source.
  • Click on Agree button if you think the analysis of news was correct.

    • Admin Panel:

  • To Add a new Source, Ennter the details of the source in top left box and click add.
  • To preview a source data, search for it in the box on top right.
  • All preview of data can be seen in the box in the second row.
  • You can edit this data by clicking edit button. Once changes are made save it by clicking save
  • You can see requests by source parties in the next row.
  • Once the request is handled, press resolve to delete the request.
